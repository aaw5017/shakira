## VirtualEnv Setup

1. Install VirtualEnv
2. Inside the root directory, `virtualenv -p <path_to_python>` .venv

## Activate the Virtual Environment
`activate-venv.bat`

## Install Pip Dependencies
`pip install -r requirements.txt`
